# AASANA AI
## Deployment link https://aasana-ai.netlify.app/
## Setup Instructions

### 1. Clone the Repository

First, clone this repository to your local machine:

```bash
git clone https://github.com/viswaprateek/AASANA_AI.git
cd AASANA_AI
```
### 2.Install Dependencies
```bash
npm i
```

### 4. Run
```bash 
npm start
```
